Caprae Capital Lead Generation Tool

Quick Start
```bash
# Clone the repository
git clone https://github.com/purviwasanji/caprae-lead-generation-tool.git

# Navigate to project directory
cd caprae-lead-generation-tool

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
Features
- Intelligent Company Analysis: 110+ companies with 40+ data points
- Acquisition Score Algorithm**: Prioritizes high-potential targets based on revenue growth, employee growth, market position
- Personalized Outreach Generator**: AI-powered messaging with conversation starters and follow-up strategies
- Interactive Dashboard**: Real-time analytics showing industry distribution, revenue vs acquisition scores
- CRM Integration: Export contacts, copy emails, and manage outreach status
- Priority Target Filtering**: Advanced filtering by acquisition score, industry, and market position
- Executive Contact Database**: Direct access to CEO/CTO contact information and LinkedIn profiles

Technical Stack
- Backend: Python
- Frontend: Streamlit
- Data Processing: Pandas, NumPy
- Visualization: Plotly, Matplotlib
- UI Framework: Streamlit components

Dataset
The tool processes 110 companies with comprehensive data including:
- Company financials and growth metrics
- Executive contact information
- Market position and competitive analysis
- Investment thesis and risk assessment

Business Impact
- Streamlined lead research with comprehensive company profiles
- Automated outreach generation reducing manual message writing
- Strategic deal prioritization through acquisition scoring
- Enhanced sales efficiency with integrated contact management
